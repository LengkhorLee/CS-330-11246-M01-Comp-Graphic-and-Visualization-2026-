# CS-330-11246-M01-Comp-Graphic-and-Visualization-2026-
Throughout this project, I learned that designing software takes a lot of planning before writing code. I started by thinking about how I wanted my scene to look and then broke it down into smaller objects that I could build one at a time. As I worked through each milestone, I made small changes and tested them before moving on. This made it easier to find problems and fix them without creating new ones. In the future, I think this approach will help me stay organized when working on larger projects.

When developing the program, I learned more about using OpenGL, textures, lighting, camera movement, and organizing my code into reusable functions. There were times when things did not work the way I expected, so I had to experiment, make adjustments, and test again until I got the results I wanted. Each milestone helped build on what I learned in the previous one, and by the end of the course I felt much more comfortable working with 3D graphics.

This project also showed me how computer graphics can be useful outside of video games. The skills I learned can be applied to simulations, engineering, product design, virtual reality, and many other fields. Since I work in IT, understanding how graphics applications are built gives me a better overall understanding of software development. It also gives me experience that I can use in future courses and could be helpful if I decide to work on graphics, game development, or other software projects later in my career.
